关于 Apple ID 无法登录的一堆破事

谜一样的无法登录

年初的时候，国区的 Apple ID 在手机登录不了 App Store。
情况是大致是登录了其他区的 ID 后，手机想切换回国区的 ID，总是回提示错误。同样的账号密码在电脑端却可以顺利登录。
手机端的错误提示包括但不限于图中两种，这边可以肯定的是，只要你确认账号密码没有问题，但还是提示你密码错误，那就是遇到了这一状况。

这一次的解决方案是重设密码，之后就没什么问题了。

各种无谓的尝试

三个月前轮到美区的 Apple ID 登录不了，本以为修改下密码也就可以了，却折腾了好几个月。

第一次尝试是修改密码／重启手机，无果。
网上搜了下，发现类似这种情况的在 iOS 8 的年代也有发生，就试了下退出重登录当前 iCloud 账号的做法，中间掺杂着各种重启，毕竟「重启试试」是人生四大金句之一。不过依然没什么效果。
恰逢 iOS Public Beta 发布，就指望升级下系统能搞定，不过依然是扯淡。
这时候想起了 Apple 的客服，预约了电话支持后，当天打了 40 来分钟的电话，客服支持说因为账号是美区的，获取不到太多信息，但可以确定的一点是账号没有被封锁。
这也是我之前担心的一点，既然没有被封锁，就应该是手机的问题了，谜一样的 Apple 账户系统。
因为当场没解决问题，客服很热心地留下了邮箱并教导了两个现在想想完全是没什么用的操作，本指望着反馈过去后就能有解决方案，等到了这几天还是没有。

重启试试

昨天加班无聊，就重新试了下。
首先是在「设置」－「iTunes Stores 与 App Store」重置 Apple ID，无果。
然后尝试在「设置」－「iCloud」重置 iCoud ID，无果。
之后尝试在「设置」－「通用」－「还原」－「还原所有设置」中抹掉设置信息，无果。
狠下心来，直接在设置」－「通用」－「还原」－「抹掉所有内容和设置」中重置手机。
经过几分钟的重启后，手机焕然一新，所有 app 都没了，包括微信聊天记录，因为我忘记备份了。
哦，欣慰的是，Apple ID 那档子破事终于搞定了。
于是，昨晚又花了几个小时下载 app，登录各种账号，得益于原来各种依赖在线备份，除了微信聊天记录，基本所有东西都在，回头有空总结下。

在这里再次温习下人生四大金句：
不行就分／喜欢就买／多喝热水／重启试试。
